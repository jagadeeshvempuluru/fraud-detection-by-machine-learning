# Ecommerce Fraud Detection using Machine Learning

This project is a simple web application that detects whether an ecommerce transaction is fraudulent or not using a machine learning model.  
I built this project to understand how ML can be integrated with a Django web application and how predictions can be shown in a user interface.

The user can enter the transaction details through the web page and the system will predict if it is a normal transaction or a fraud.

---

## Why I built this project

Online fraud is increasing day by day and I wanted to learn how machine learning can help in solving real-world problems.  
While doing this project I learned:

- How to train a model using a dataset
- How to connect the ML model with Django
- How to take input from the user and show the prediction result

## How to run this project on your system

1. Clone the repository

   git clone https://github.com/jagadeeshvempuluru/fraud-detection-by-machine-learning.git

2. Go to the project folder

   cd fraud-detection-by-machine-learning

3. Create a virtual environment

   python -m venv venv

4. Activate the environment

   venv\Scripts\activate

5. Install the required packages

   pip install -r requirements.txt

6. Run the server

   python manage.py runserver

7. Open in browser

   http://127.0.0.1:8000/

## Project workflow

- Load the dataset
- Train the machine learning model
- Save the trained model
- Take user input from the web page
- Send the data to the model
- Display whether the transaction is fraud or not
